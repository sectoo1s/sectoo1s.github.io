typora的使用方法

1、标题

“#”符号为标题的等级，“#”符号越少标题越大，最多可以连续写六个“#”符号。输入“#”符号之后一定要加一个空格然后继续写，不然不会产生样式效果。

2、字体

加粗：在字体两边分别加两个星号。

斜体：在字体两边加一个星号。

粗体加斜体：在字体两遍分别加三个星号。

删除线：在字体两边分别加两个(～)波浪线。

3、引用

在起始位置添加一个大于符号(>) + 空格，表示引用。

一给我我里giaogiao！——giao哥

4、分割线

可以用三个杠(_) + 空格

也可以用三个星号（*）+空格（这个分割线占全屏）

5、图片

样式（英文状态下的感叹号 加 中括号 加 小括号（ !+[ ]+()），中括号里面放图片的名字，小括号放图片的地址（图片可以是本地的图片也可以是网上的图片））。

6、超链接

样式：（英文状态下的中括号 + 小括号（[ ]+( )）,中括号放超链接的名字，小括号放超链接的地址）

百度一下测试

7、列表

有序列表：样式(1 + .号 + 空格)

  1. a
  2. b
  3. c

无序列表：样式(- 号 + 空格)

  • a
  • b
  • c

8、表格

Mac下快捷键插入：option + command + t

9、代码

打开Typora后在编辑区输入三个点号，该键位在esc按键下方数字1的前面。然后输入html，注意要挨着三个点号不要加空格。回车后代码块出现，可在代码块中输入javascript代码，代码块右下角有指示当前语言为javascript。
